# warehousemanager
A warehouse management application written in Java. Contributors Miska Peltoniemi, Marko Korhonen, Jere Moilanen and Huy Nguyen.

# Technical details
* The application is built with Java and JavaFX
* Data is stored with MySQL
* Database layer is using Hibernate
